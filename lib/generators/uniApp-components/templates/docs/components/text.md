# Text 文本

用于包裹文本内容。

<Mobile src="http://front-end.pages.cloudhw.cn:8000/ember/example/#/v-text"/>

## 代码演示

### 基础使用

文字内容通过 `slot` 传入

```vue
<v-text>这是一个文本</v-text>
```

### 自定义字体颜色

```vue
<v-text color="red">自定义字体颜色</v-text>
```

## API

### Props

| 参数  | 说明 | 类型   | 默认值 |
| ----- | ---- | ------ | ------ |
| color | 颜色 | string | -      |
